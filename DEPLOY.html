Deploy Lens

Free hosting · Any computer · 5 minutes · No install needed on any machine

✅ Once deployed, anyone can use Lens by visiting a URL. No setup on their end. Works on school computers, library computers, phones — anything with a browser.

1 Get your Anthropic API Key free to start

Go to console.anthropic.com → Sign up (free) → Click API Keys → Create a new key

Copy the key — it looks like: sk-ant-api03-…

Anthropic gives you free credits to start. Claude Haiku costs about $0.001 per message — very cheap.

2 Upload the project to GitHub free

1. Create a free account at github.com

2. Click New repository → name it lens → Public → Create

3. Upload all the files from the lens/ folder you downloaded (drag them in via the GitHub web UI):

lens/ ├── package.json ├── backend/ │ └── server.js └── frontend/ └── index.html

Make sure the folder structure stays the same when uploading.

3 Deploy on Render.com free tier available

1. Go to render.com → Sign up with your GitHub account

2. Click New + → Web Service

3. Connect your GitHub repo (lens)

4. Fill in these settings:

Name: lens (or anything you want) Runtime: Node Build cmd: npm install Start cmd: npm start Branch: main

5. Scroll to Environment Variables → click Add Environment Variable:

Key: ANTHROPIC_API_KEY Value: sk-ant-api03-your-actual-key-here

6. Click Create Web Service → wait ~2 minutes for it to build

4 Get your URL and share it

Render gives you a free URL like:

https://lens-abcd.onrender.com

That's it. Open that URL on any computer — it just works.

Bookmark it. Share it with yourself. Works on any device with a browser.

⚠ Free tier note: Render's free tier "sleeps" after 15 minutes of inactivity. The first load after sleeping takes ~30 seconds to wake up. To avoid this, upgrade to Render's $7/month plan, or use Railway.app which has a more generous free tier.

Alternative: Railway.app

If you prefer Railway (better free tier, faster cold starts):

1. Go to railway.app → Login with GitHub

2. New Project → Deploy from GitHub → select lens

3. Add variable: ANTHROPIC_API_KEY = your key

4. Railway auto-detects Node and deploys. Done.

✅ Your API key stays secret on the server. Nobody visiting the site can see it. All they see is the chat interface. You pay for the API calls but the key is safe.
