⚡ How to deploy (free, 5 minutes)

1. GitHub — Push this project to a GitHub repo (free account)
2. Render.com — Connect your repo, set ANTHROPIC_API_KEY as an environment variable, deploy
3. Done — You get a public URL like https://lens-xyz.onrender.com that works on any computer, anywhere, forever

Your API key stays secret on the server — nobody else can see it. Users just visit the URL.
Cost: Render free tier is enough for personal use. Anthropic charges per message (~$0.001 per chat with Haiku).
